## 2. Guiding principles and constraints

These rules apply across every phase and every workstream:

1. **Human-centric AI** — AI augments operational teams; it does not replace annotators, SMEs, or scientific reviewers.
2. **Evidence-backed AI** — Every AI insight and client narrative links to citable source data. No hallucinated project status.
3. **Tenant isolation** — Client users never see another organisation's data. Enforced by PostgreSQL RLS, application-layer scoping, and mandatory Playwright e2e tests.
4. **Human-in-the-loop for outbound client communications** — 100% of AI-drafted client communications require explicit Delivery Manager approval before `sent`.
5. **Synthetic data only in non-prod** — `dev` and `staging` use sanitised seed data until governance sign-off. Production requires client approval, governance sign-off, security validation, and compliance review.
6. **Admin-configurable client metrics** — Super Admin controls client-facing metrics via `metric_configurations`; not hardcoded.
7. **EU & client-region data residency** — Supabase region selection is irreversible; must be decided before production provisioning.

### Confirmed technology choices (binding)

| Layer | Choice |
|---|---|
| Frontend | React 18, TypeScript, Tailwind CSS, shadcn/ui, Zustand, TanStack Query, Vite, Recharts |
| Backend | Python 3.12, FastAPI, SQLAlchemy async + asyncpg |
| Database | PostgreSQL via Supabase; RLS; Supabase CLI migrations |
| Auth | Supabase Auth + JWT; RBAC via RLS |
| Repo | Monorepo: `frontend/`, `backend/`, `supabase/`, `e2e/` |

**Unresolved (block agent code):** LLM provider/model (DQ-021), vector/RAG store (DQ-022), backend hosting (DQ-034).

### Draft quality thresholds (WIP — not contractual)

Read from `metric_configurations` at runtime, not hardcoded:

| Metric | Draft threshold | Direction |
|---|---|---|
| On-time milestone delivery | ≥ 90% | Portfolio KPI |
| Schedule confidence (per milestone) | ≥ 80% | Below = `at_risk` |
| Gold-set accuracy | ≥ 95% | Below triggers drift alert |
| Inter-annotator agreement (Krippendorff's α) | ≥ 0.85 | Below triggers drift alert |
| Rework rate | ≤ 5% | Above triggers drift alert |
